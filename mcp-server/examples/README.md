# Examples

This directory contains example scripts demonstrating how to use the SIENNA-PA-MCP server.

## Available Examples

### basic_usage.py

Demonstrates basic usage of the MCP server including:
- Loading configuration
- Listing available tools
- Executing a simple tool

## Running Examples

To run an example:

```bash
cd mcp-server/examples
python basic_usage.py
```

Make sure you have installed the package first:

```bash
cd mcp-server
pip install -e .
```
